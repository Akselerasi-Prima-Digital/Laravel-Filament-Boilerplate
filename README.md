# Template Laravel Filament

Template ini adalah starter kit untuk aplikasi Laravel yang sudah terintegrasi dengan:
- **Filament Admin Template** menggunakan [Filament](https://filamentphp.com/)
- **Manajemen Permission & Role** menggunakan [Spatie Laravel Permission](https://spatie.be/docs/laravel-permission/v6/introduction)

## Fitur
- Manajemen user dengan role-based access control (RBAC)
- Direct permission untuk user tanpa perlu role
- Autentikasi: Login, Lupa Password, Verifikasi Email
- Tampilan dashboard yang modern menggunakan Filament
